动态sql

|          元素           |         作用          |                    备注                    |
| :---------------------: | :-------------------: | :----------------------------------------: |
|           if            |       判断语句        |               单条件分支判断               |
| choose，when，otherwise | 相当于java的case when |               多条件分支判断               |
|    trim，where，set     |       辅助元素        |            用于处理sql拼装问题             |
|         foreach         |       循环语句        | 在in语句等列举条件常用，常用于实现批量操作 |
